# AlgoSaathi 🧠💻

**AlgoSaathi** is an AI-powered coding platform designed to help developers write, debug, and optimize code in real-time. It features a built-in code editor, multi-language support, and an intelligent AI tutor that provides explanations, hints, and debugging assistance.

## 🚀 Live Demo - https://algo-saathi.vercel.app/

## ✨ Features
- **AI-Powered Assistance:** Get instant explanations, hints, and debug help using Gemini AI.
- **Multi-Language Support:** Run C++, Python, Java, and JavaScript directly in the browser.
- **User Authentication:** Secure login/signup with JWT and Google OAuth.
- **Profile & Analytics:** Track your coding streak, submission history, and activity heatmap.
- **Real-time Execution:** Powered by the Piston API for safe and fast code execution.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Monaco Editor
- **Backend:** Node.js, Express.js, MongoDB
- **AI Engine:** Google Gemini API
- **Authentication:** JWT, Google OAuth 2.0
- **Deployment:** Vercel (Frontend), Render (Backend), MongoDB Atlas
